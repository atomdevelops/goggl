### Features

- Links on search
- View all searches
- View related News
- View images
- View videos

## Installation

**This is only for those who want to run it on their own machines**

Clone the GitHub repository.

```bash
git clone https://github.com/atomdevelops/goggl.git
cd goggl--master
```

Install required dependencies

```bash
npm i
# or with yarn
yarn
```

Run the application

```bash
npm start
# or with yarn
yarn start
```
